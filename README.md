### 简介
一个完整的录音功能dialog（当然在Activity中实现更加简单），使用dialogfragment实现，并解决dialogfragment在展示上所遇到的一些坑点，见效果图。

并附带了波形音频效果，如与你们的设计图效果不符合，可简单操作更改到自己满意的UI效果。
